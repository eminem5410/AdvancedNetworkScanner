🌐 Advanced Network Scanner Pro
Una herramienta de administración de redes profesional, moderna y de código abierto para Windows. Diseñada para escaneos rápidos, detección de dispositivos y administración remota.

Icono

✨ Características Principales
Este escáner no solo lista direcciones IP; es una completa herramienta de gestión de red.

🔍 Escaneo Avanzado
Auto-Detección de Subred: Detecta automáticamente tu red local (192.168.x.x, 10.0.x.x, etc.) al iniciar.
Rango Personalizable: Permite escanear rangos completos de IP (ej: 10.10.10.1 hasta 10.10.10.255).
Escaneo Paralelo Rápido: Optimizado para escanear cientos de dispositivos en segundos.
IP Pública Detectada: Muestra automáticamente la IP externa de tu conexión.
🏷️ Inteligencia de Dispositivos
Identificación por MAC: Resuelve direcciones MAC para determinar el fabricante.
Clasificación Automática: Identifica el tipo de dispositivo visualmente:
🔴 Router/AP (Routers y Puntos de Acceso)
🔵 Smartphone/Tablet (Apple, Samsung, Xiaomi, etc.)
🟡 Smart TV (Samsung, LG, Sony, Hisense)
⚪ PC/Laptop (Intel, Realtek)
Ordenamiento Inteligente: Los dispositivos "Online" se ordenan automáticamente arriba de la lista.
Estado en Tiempo Real: Indicadores de Latencia (ms) y estado (Online/Offline) con colores.
⚡ Acciones Remotas (Doble Clic en Dispositivo)
Al hacer doble clic en cualquier dispositivo de la lista, se abre un panel de acciones:

🌐 Navegador Web: Abre http://[IP] automáticamente en tu navegador.
📁 Carpetas Compartidas (SMB): Abre el Explorador de Windows en \\[IP].
⚡ Wake-on-LAN (WOL): Envía "Magic Packets" para encender remotamente PCs apagadas (requiere MAC válida y BIOS configurada).
💻 Conexión Remota (RDP): Lanza mstsc.exe para conectarte vía Escritorio Remoto.
🔌 Escáner de Puertos: Escanea automáticamente puertos comunes (FTP, SSH, Telnet, HTTP, HTTPS, SMB, MySQL, PostgreSQL, VNC, etc.) y muestra los abiertos.
📋 Copiar IP: Copia la dirección IP al portapapeles.
📊 Gestión de Datos
Exportar a Excel (CSV): Genera un reporte detallado (IP, Hostname, MAC, Status, Latency) compatible con Microsoft Excel con un solo clic.
🎨 Interfaz y Experiencia de Usuario
Tema Oscuro Profesional: Interfaz inspirada en IDEs modernos (VS Code / Visual Studio) para reducir la fatiga visual.
Indicador de Progreso: Barra de carga visual mientras escanea la red.
Icono Personalizado: Identidad visual única.
Firma del Desarrollador: Créditos visibles en la aplicación.

🚀 Instalación y Uso
Descarga la última versión en la sección Releases.
